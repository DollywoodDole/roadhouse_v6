# Stripe Products & Price IDs

All prices in **CAD**. All price IDs generated by `scripts/create-stripe-prices.ts`.

To regenerate after a price change (archive old price in Stripe first):
```bash
STRIPE_SECRET_KEY=sk_live_... npx ts-node --project tsconfig.scripts.json scripts/create-stripe-prices.ts
```

## Notes

- **Stripe account:** `acct_1TA4dCDv5Ly4OO7E`
- **Currency:** All CAD
- **Idempotency:** Each price has `metadata.rh_env_var` set — re-running the script skips existing prices
- **Price updates:** Stripe prices are immutable. Archive in dashboard → re-run script to get new ID → update Vercel env
- **Webhooks:** `STRIPE_WEBHOOK_SECRET` in Vercel must match the signing secret for `https://roadhouse.capital/api/webhooks/stripe`
